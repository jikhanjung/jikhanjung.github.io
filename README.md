# Jikhan Jung's Portfolio

Personal website showcasing paleontology software projects.

**Live Site**: [https://jikhanjung.github.io](https://jikhanjung.github.io)

## About

Paleontologist and software developer creating tools for:
- Geometric morphometrics
- Phylogenetic analysis
- CT image processing
- Data visualization

## Projects

- **[Modan2](https://jikhanjung.github.io/Modan2/)** - Comprehensive geometric morphometrics software
- **[PhyloForester](https://jikhanjung.github.io/PhyloForester/)** - GUI shell for phylogenetic analysis using TNT/IQTree/MrBayes
- **[CTHarvester](https://jikhanjung.github.io/CTHarvester/)** - CT image stack preprocessing software

## Technology

This site is built with:
- Jekyll (static site generator)
- GitHub Pages (hosting)
- `jekyll-theme-hacker` theme

## Local Development

```bash
# Install dependencies
gem install bundler jekyll

# Serve locally
jekyll serve
```

Visit `http://localhost:4000` to preview changes.
